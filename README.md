# Chatpal MVP v1.0

Chatpal is a rule-based booking assistant built using Python.

## Features
- Service price detection
- Booking system
- Customer data collection
- JSON-based storage
- Admin command to view leads

## Example
Customer: book x-ray  
Chatpal: Booking confirmed for x-ray at rs 700

## Tech Stack
- Python
- JSON
- Loop-based conversation engine

This is an early backend prototype of a SaaS AI booking assistant for small businesses like clinics, salons, gyms, coaching centres or restaurants.
